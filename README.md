# Проект сайт для магазина книг на django

## Главная страница до авторизации
3 активные кнопки: 

"Начать покупки" и "Каталог" ведут на одну страницу - в каталог книг

"Войти" - ведут на страницу авторизации

## Страница авторизации
Стандартная форма django для ввода логина и пароля. По кнопке регистрации переходит на страницу с регистрацией с django формой с логином, паролем и подтверждением пароля.

В качестве бд используется sqlite

Таблица - стандартная от django - auth_users

## Главная страница после авторизации
добавляются кнопки коризна и мои заказы

## Каталог
В каталоге представлены книги, которые хранятся в бд в таблице myapp_book с полями: id, title, author, description, price, cover_image.

На страницу каждой книги можно зайти и добавить в корзину.

## Корзина
В корзине хранятся книги, которые туда были добавлены.

Книги можно убрать из корзины.

Также можно сделать заказ

Тут используются 2 таблицы из бд: myapp_cart с полями id, created_at, updated_at, user_id и myapp_cartitem с полями id, quantity, book_id, cart_id

## Заказ

В странице с заказом нужно указать ФИО, email, телефон и адрес доставки. После этого можно сделать заказ.

Тут тоже используются 2 таблицы: myapp_order и myapp_orderitem

## Деплой

Сайт разворачивал с помощью docker-compose на своём сервере с открытыми портами
